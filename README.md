# Whimsical Cursor Plugin

A [Cursor](https://cursor.com) plugin that provides access to Whimsical workspaces, enabling AI assistants to create and edit boards, flowcharts, mind maps, wireframes, docs, and more.

## Overview

This plugin connects your AI assistant to Whimsical.

## Installation

1. Open Cursor settings
2. Navigate to **Plugins**
3. Click **Browse Marketplace**
4. Search for "Whimsical"
5. Click **Install**
